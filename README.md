# PDF2MD - PDF to Markdown Converter (v3.3)

PDFファイルをMarkdown形式に変換するWindowsデスクトップアプリケーションです。

**PyMuPDF**、**PyMuPDF4LLM**、**EasyOCR** を使用して、テキスト・表・画像を高精度に抽出し、文書構造を保持したMarkdown変換を実現します。

## 主な機能

### 基本機能
- **テキスト抽出**: PDFからテキストを位置情報付きで抽出
- **画像抽出**: PDF内の図・画像を自動的に抽出して保存（ラスター画像＋ベクター描画対応）
- **ベクター描画検出** (v3.1新機能): ラスター画像がないPDFでもベクター描画（図形・イラスト・背景）を検出してPNG画像として抽出
- **PyMuPDF4LLM統合** (v3.2新機能): レイアウト認識変換エンジンを搭載、マルチカラムPDFの読み順を正確に保持
- **OCR対応**: 図内のテキストをOCRで認識（日本語・英語対応）
- **位置保持**: テキストと画像と表の位置関係を正確にMarkdownに反映

### 高度な構造認識 (v3.0新機能)
- **高度な表検出**: 罫線あり/なし両方の表を検出してMarkdownテーブル形式に変換
- **見出し階層自動認識**: 文書全体のフォントサイズを分析して見出しレベル（H1-H6）を自動判定
- **リスト構造認識**: 箇条書き・番号付きリストの階層を保持
- **図表キャプション対応**: 「図1」「表1」などのキャプションを自動検出して紐付け
- **罫線なし表検出**: テキスト位置から表構造を推定（データ一覧など）

### マルチカラム対応 (v3.2新機能)
- **PyMuPDF4LLM優先変換**: 利用可能な場合、レイアウト認識エンジンで変換（2カラム対応）
- **ベクター描画自動補完**: PyMuPDF4LLM変換後にベクター描画画像を自動検出・挿入
- **重複画像排除**: PyMuPDF4LLMとベクター描画の重複画像を自動検出して高品質版に統合
- **フォールバック変換**: PyMuPDF4LLM失敗時は標準パイプラインに自動切替（2カラム検出付き）

### 使いやすさ
- **GUI対応**: 直感的なグラフィカルインターフェース
- **ドラッグ&ドロップ**: PDFファイルをウィンドウにドロップして追加
- **バッチ処理**: 複数ファイルの一括変換
- **CLIモード**: コマンドラインからも使用可能

## インストール

### 方法1: EXEファイルを使用（推奨）

1. [Releases](https://github.com/atsushisuemitsu/PDF2MD/releases) から最新の `PDF2MD.exe` をダウンロード
2. ダウンロードしたEXEを実行

### 方法2: Pythonから実行

```bash
# リポジトリをクローン
git clone https://github.com/atsushisuemitsu/PDF2MD.git
cd PDF2MD

# 依存関係をインストール
pip install -r requirements.txt

# 実行
python pdf2md.py
```

## 使い方

### GUIモード

1. `PDF2MD.exe` をダブルクリックして起動
2. 「ファイル追加」または「フォルダ追加」でPDFを選択
   - またはPDFファイルをウィンドウにドラッグ&ドロップ
3. オプションを設定:
   - **画像を抽出**: チェックするとPDF内の画像を別ファイルとして保存
   - **OCR**: チェックすると画像内のテキストをOCRで認識してMarkdownに含める
4. 必要に応じて出力先を設定
5. 「変換実行」をクリック

### 右クリックメニューから変換（Windows）

エクスプローラーでPDFファイルを右クリックするだけで変換できます。

#### セットアップ

1. `install_context_menu.bat` を**管理者として実行**
2. 完了メッセージが表示されれば登録成功

#### 使い方

1. エクスプローラーでPDFファイルを右クリック
2. **「PDF → MD 変換」** を選択
   - Windows 11では「その他のオプションを確認」から表示されます
3. コンソールウィンドウが開き変換が実行されます
4. 変換完了後、PDFと同じフォルダに `.md` ファイルと `_images` フォルダが生成されます

#### アンインストール

`uninstall_context_menu.bat` を管理者として実行するとメニューが削除されます。

### CLIモード

```bash
# 単一ファイル変換（画像抽出あり）
python pdf2md.py document.pdf

# 複数ファイル変換
python pdf2md.py file1.pdf file2.pdf file3.pdf

# フォルダ内の全PDFを変換
python pdf2md.py ./pdf_folder/

# OCRを有効にして変換
python pdf2md.py --ocr document.pdf

# 画像抽出を無効にして変換
python pdf2md.py --no-images document.pdf
```

## 技術仕様

- **Python**: 3.9+

- **レイアウト認識**: PyMuPDF4LLM（マルチカラム対応）
- **OCR**: EasyOCR（日本語・英語対応）
- **画像処理**: Pillow
- **GUI**: tkinter
- **D&D対応**: tkinterdnd2（オプション）
- **EXE化**: PyInstaller

## 変換の仕組み

### 変換パイプライン（2段階）

#### 優先: PyMuPDF4LLM変換
1. **レイアウト認識変換**: PyMuPDF4LLMでマルチカラム対応のMarkdown変換
2. **ベクター描画補完**: 変換後にベクター描画画像（章タイトルページ等）を自動検出・挿入
3. **重複排除**: PyMuPDF4LLMが抽出済みの画像と重複するベクター描画は高品質版に置換

#### フォールバック: 標準パイプライン

### Stage 1: 文書構造分析
1. **PDF解析**: PyMuPDFでPDFを開き、各ページを解析
2. **フォント分析**: 文書全体のフォントサイズを収集・分析して見出し階層を推定
3. **本文サイズ特定**: 最も頻出するフォントサイズを本文として認識

### Stage 2: 要素抽出
4. **高度な表検出**: PyMuPDFの表検出 + テキスト位置ベースの罫線なし表検出
5. **テキストブロック抽出**: 表領域を除外してテキストを抽出
6. **ラスター画像抽出**: 埋め込みビットマップ画像をPNGファイルとして保存（50px未満のアイコンは除外）
7. **ベクター描画抽出**: ページ上のベクター描画要素をクラスタリング→フィルタリング→PNG画像としてレンダリング
8. **OCR処理**: EasyOCRで図内のテキストを認識

### Stage 3: 構造認識
9. **見出し判定**: フォントサイズ・ボールド属性から見出しレベル（H1-H6）を決定
10. **リスト認識**: マーカーパターン（●○・1.など）からリスト構造を検出
11. **テキスト重複除外**: フルページ描画画像に含まれるテキストブロックを除外
12. **2カラム検出**: X座標分布からマルチカラムレイアウトを検出して読み順を補正
12. **キャプション紐付け**: 「図1」「表1」などを最も近い図表と関連付け

### Stage 4: Markdown生成
13. **位置ソート**: 全要素をページ順→Y座標順→X座標順でソート
14. **Markdown出力**: 構造情報を保持したMarkdown形式で出力

## 注意事項

- OCR機能を使用する場合、初回実行時にモデルファイル（約100MB）がダウンロードされます
- 大きな画像を含むPDFの場合、処理に時間がかかることがあります
- 複雑なレイアウトのPDFでは、位置関係が正確に再現されない場合があります
